# Управление промтами

## Применение миграции

1. Откройте Supabase SQL Editor
2. Выполните SQL из файла `app/db/migrations/add_user_prompts.sql`

## Использование

### Управление промтами

1. Запустите приложение
2. Подключитесь к БД (F5)
3. Меню **Настройки → Промты** (Ctrl+P)

### Применение промтов в чате

1. В нижней панели чата выберите промт из выпадающего списка
2. System prompt применится автоматически
3. User text (если есть) заполнит поле ввода
4. Кнопка ✏️ открывает редактор выбранного промта

## Функционал

- **Создание** промтов с system_prompt и user_text
- **Редактирование** существующих промтов
- **Удаление** промтов
- **Автосохранение** на R2 и в БД
- **Список** всех промтов пользователя
- **Выбор промта** в чате перед отправкой запроса
- **Быстрое редактирование** из чата
